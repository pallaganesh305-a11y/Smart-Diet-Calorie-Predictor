# Smart Diet Calorie Predictor

This project is an end-to-end machine learning application that predicts calorie content based on nutritional values.

## Features
- Predicts calories using Random Forest Regressor
- Built using Python, Pandas, Scikit-learn
- Deployed using Streamlit

## Input Features
- Protein
- Carbohydrates
- Fat
- Fiber
- Sugars
- Sodium
- Cholesterol
- Water intake

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   streamlit run app.py
